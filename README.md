# 🛒 ShopEase

A modern Django-based E-Commerce web application where users can browse products, add items to their shopping cart, place orders, and manage their shopping experience through a clean and responsive interface.

---

## 🚀 Features

- 🔐 User Registration & Login
- 🛍️ Product Catalog
- 🛒 Shopping Cart
- ➕ Add to Cart / Remove from Cart
- 💳 Checkout System
- 📦 Order History
- 🖼️ Product Image Support
- 👤 User Authentication
- ⚙️ Django Admin Panel
- 📱 Responsive & Modern UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Ashish4chauhan/ShopEase-Ecommerce.git
```

### Go to Project Folder

```bash
cd ShopEase-Ecommerce
```

### Create Virtual Environment

```bash
python -m venv env
```

### Activate Environment

**Windows**

```bash
env\Scripts\activate
```

**Linux / macOS**

```bash
source env/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Database Migrations

```bash
python manage.py migrate
```

### Start Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
